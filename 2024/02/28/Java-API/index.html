Math

Math包含执行基本数字运算的方法，如基本指数，对数，平方根和三角函数。
方法：
abs 取绝对值
ceil 向上取整
floor 向下取整
round 四舍五入
max 两值取大
min 两值取小
pow 幂
random
log 以十为底

system

exit(0),正常停止Java虚拟机
currentTimeMillis​() 返回当前时间（以毫秒为单位）。
gc () 运行垃圾回收器。
arraycopy​(Object src, int srcPos, Object dest, int destPos, int length) 将指定源数组中的数组从指定位置复制到目标数组的指定位置

Objects

static String toString ( Object o) 返回非null参数调用toString的结果和null参数的”null”的null 。 调用对象的toString方法
static String toString​(Object o, String nullDefault) 如果第一个参数不是 null ，则返回第一个参数上调用 toString的结果，否则返回第二个参数。
static boolean isNull​(Object obj) 返回 true如果提供的引用是 null否则返回 false 。
static boolean nonNull​(Object obj) 返回 true如果提供的是非null否则返回 false 。

BigDecimal

java运算时会将十进制先转为二进制计算（小数会有数据丢失），再转为十进制展示
BigDecimal() 传递小数或字符串
用来进行精确计算（使用字符串的构造方式）
常用方法
add
subtract
multiply
divide(BigDecimal divisor, int scale, RoundingMode roundingMode) 表示精确到多少位和舍入模式
ROUND_HALF_UP 四舍五入
ROUND_UP 进一
ROUND_FLOOR 去尾

##时间日期类

Date

一个精确到毫秒的特定时间
构造函数可以以long为参数，为从计算机的时间原点开始加上这个毫秒值的时间；0L：1970年1月1日早上8点（北京时区）
空参的时间为电脑中的当前时间
long getTime()
void setTime()

SimpleDateFormat

yyyy-MM-dd HH:mm:ss
yyyy年MM月dd日 HH:mm:ss

构造方法
(String pattern);
格式化：String format(date);
解析：Date parse(String s)

静态方法：
now() 当前时间
of(年，月，日，时，分，秒)

int getDayOfMonth​() 获取几号。
DayOfWeek getDayOfWeek​() 获取星期几字段，这是一个枚举 DayOfWeek 。
int getDayOfYear() 获得一年中的第几天。
int getHour​() 获取时间字段。
int getMinute() 获取小时字段。
Month getMonth() 使用 Month枚举获取月份字段，获得对应英文。
int getMonthValue​() 获得月份。
int getYear() 获取年份字段。

LocalDate toLocalDate​() 获得这个日期时间的 LocalDate部分。
LocalTime toLocalTime​() 获取此日期时间的 LocalTime部分

DateTimeFormatter

Period

static Period between(LocalDate startDateInclusive, LocalDate endDateExclusive) 获得一个 Period包括两个日期之间的年数，月份和日期。
int getDays() 获得此期间的天数。
int getMonths​() 获取此期间的月数。
int getYears() 获得这段时间的年数。
long toTotalMonths​() 获得此期间的总月数。

Duration

static Duration between​(Temporal startInclusive, Temporal endExclusive) 获取表示两个时间对象之间的持续时间的 Duration
long toDays​() 获取此持续时间的天数。
long toDaysPart​() 提取持续时间的天数。
long toHours​() 获取此持续时间内的小时数。
int toHoursPart​() 提取持续时间部分的小时数。
long toMillis​() 将此持续时间转换为总长度（以毫秒为单位）。
int toMillisPart​() 提取持续时间部分的毫秒数。
long toMinutes​() 获取此持续时间的分钟数。
int toMinutesPart​() 提取持续时间部分的分钟数。
long toNanos​() 将此持续时间转换为以 long表示的总长度，表示为 long 。
int toNanosPart​() 在持续时间的几秒钟内获取纳秒部分。
long toSeconds() 获取此持续时间内的秒数。
int toSecondsPart​() 提取持续时间中的秒数。
