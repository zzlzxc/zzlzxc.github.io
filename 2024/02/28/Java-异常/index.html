Java 异常

异常

异常就是程序出现了不正常的情况，会导致JVM的非正常停止
异常类：Exception

  • 运行时异常
  • 编译时异常

代码出现异常后，会在此处创建一个异常对象，如果程序中没有自己处理异常的代码，会交给虚拟机默认处理。JVM会将红色异常信息展示在控制台上，在此处停止程序运行。

throws

声明的异常为编译时异常时必须在方法后用throws手动声明。

throw

手动抛出运行时异常反馈给调用者。

try…catch

可以让程序继续往下运行。
没有遇到异常会把try中的代码全部执行，不执行catch
遇到异常跳过try剩下的代码去执行catch语句
异常没有被捕获会交给处理机处理

Throwable

String getMessage​() 返回此throwable的详细消息字符串。
String toString​() 返回此可抛出的简短描述。
void printStackTrace​(PrintWriter s) 将此throwable和其追溯打印到指定的打印作者。
