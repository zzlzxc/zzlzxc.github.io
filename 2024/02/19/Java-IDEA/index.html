常用快捷键

psvm：main方法
sout：输出语句
list.for：增强for循环
alt+1：打开/隐藏工程目录
alt+4：打开/隐藏控制台
alt+insert：快捷创建
alt+enter：代码修正提示
ctrl+D：向下复制一行
ctrl+X：剪切当前一行
ctrl+/：批量单行注释
ctrl+shift+/：批量多行注释
ctrl+shift+上：向上移动一行
ctrl+shift+下：向下移动一行
ctrl+alt+v：快速生成接收方法返回值的变量
ctrl+alt+t：快速包围
ctrl+alt+l：格式化代码
ctrl+alt+m：抽取为方法
